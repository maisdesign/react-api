## Descrizione

App React che recupera e visualizza una lista di attori/attrici da API esterne.
Progetto realizzato durante il corso **Boolean Fullstack Developer** (WDPT-11).

---

## Endpoint disponibili

| Endpoint | URL |
|----------|-----|
| Lista Attrici | `https://lanciweb.github.io/demo/api/actresses/` |
| Lista Attori | `https://lanciweb.github.io/demo/api/actors/` |

---

## Milestone

### Milestone 1
Al caricamento dell'applicazione, recuperare la lista degli attori o delle attrici dall'API e stamparla in console.

### Milestone 2
Preparare una card per ciascun attore/attrice con le seguenti informazioni:
- Nome
- Anno di nascita
- Nazionalità
- Biografia
- Immagine
- Riconoscimenti

### Milestone 3
Mostrare in pagina una card per ciascun attore o attrice, con grafica a piacimento.
